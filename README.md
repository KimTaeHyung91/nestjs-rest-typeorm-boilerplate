# NestJS + Rest API + TypeORM(v0.3) Boilerplate

## Description

NestJS + Rest API + TypeORM(v0.3) Boilerplate

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
